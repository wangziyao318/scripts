# Scripts

VPS scripts for personal use.

```bash
export EMAIL= DOMAIN= CF_Token= CF_Zone_ID= SSH_PUBLIC_KEY= &&
    curl -fsSL https://raw.githubusercontent.com/wangziyao318/scripts/refs/heads/main/script.sh | bash

    wget -qO- https://raw.githubusercontent.com/wangziyao318/scripts/refs/heads/main/script.sh | bash
```
